# Hitayu AI - Frontend (React)
This is the user interface for the Hitayu AI Health Assistant, developed as part of the **Code Unnati** program.

## Key Features
- **Symptom Selection:** Users can select multiple symptoms from a responsive grid.
- **AI Integration:** Connects to a FastAPI backend powered by a Random Forest Machine Learning model.
- **Instant Results:** Provides disease prediction, specific dosages, and treatment advice.

## Technologies Used
- **React.js & TypeScript**
- **Lucide-React** (for icons)
- **Axios** (for API communication)
